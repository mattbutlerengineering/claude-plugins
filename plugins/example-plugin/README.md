# Example Plugin

This is an example plugin to demonstrate the basic structure of a Claude Code skill.

## Description

A simple example plugin that demonstrates the basic structure and format for Claude Code skills.

## Usage

This example shows how to organize and document your custom plugins.

## Files

- `README.md` - This documentation file

## Implementation Notes

Claude Code skills can be implemented as shell scripts or other executable files. Refer to the official Claude Code documentation for detailed implementation guidelines.

## Customization

To create your own plugin based on this example:

1. Copy this directory structure
2. Rename the directory to match your plugin name
3. Implement your plugin functionality
4. Update this README with your plugin's documentation
